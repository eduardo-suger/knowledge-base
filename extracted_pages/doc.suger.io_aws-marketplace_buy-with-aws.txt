Buy with AWS
Display and sell AWS Marketplace products on sellers’ own websites.
Overview
​
Buy with AWS
enables AWS Marketplace sellers to display and sell AWS Marketplace products on sellers’ own websites. AWS Marketplace sellers can integrate with AWS Marketplace Call-To-Action buttons and APIs to build AWS-powered software discovery and procurement experiences on seller-owned websites. These include web properties owned by AWS Partners, including independent software vendors (ISVs), Channel Partners (CPs), Distributors (DPs).
warning
At present, only SaaS product listings with a subscription pricing model are supported for
Buy with AWS
in the AWS Marketplace.
Other types of product listings and pricing models will be made available for
Buy with AWS
in the future, following the AWS Marketplace release roadmap.
Buy with AWS
​

Buy with AWS
